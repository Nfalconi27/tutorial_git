Testando o git

Aprendendo o git

git init --> inicia o git na máquina

git add . --> adiciona todos os arquivos e pasta no git local

git commit -a -m "mensagem" --> criou-se uma nova versão do programa, alterando os arquivos

ir até o github, criar repositório, depois fazer os comandos para integrar o git local com o remoto

git checkout -b "nome da branch" --> cria nova branch

git push --> quando é em uma branch recém criado, surge um comando para incluir a branch no git do repositório

git checkout main depois git merge teste--> unir branch com main

git pull --> busca os arquivos do repósitorio para o local
